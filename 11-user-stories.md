# User Stories

## Authentication

**US-01**  
As a student, I want to register an account using my university email so that I can access the platform.

**US-02**  
As a registered user, I want to log in with my email and password so that I can manage my listings and messages.

**US-03**  
As a logged-in user, I want to log out so that my account stays secure on shared devices.

---

## Posting Listings

**US-04**  
As a student, I want to post a "Lost" listing with a photo, description, item category, and campus location so that others can help me find my item.

**US-05**  
As a student, I want to post a "Found" listing with a photo, description, item category, and campus location so that the owner can identify and claim their item.

**US-06**  
As a poster, I want to edit my listing after posting so that I can fix mistakes or add more details.

**US-07**  
As a poster, I want to delete my listing if the item situation is resolved outside the platform.

---

## Browsing and Searching

**US-08**  
As a student, I want to browse all active listings on the board without needing to log in so that I can quickly check if my item has been found.

**US-09**  
As a student, I want to search listings by keyword so that I can find specific items quickly.

**US-10**  
As a student, I want to filter listings by category and campus location so that I see only relevant results.

---

## Contacting and Resolving

**US-11**  
As a student, I want to send a message to the poster of a listing so that I can claim an item or provide more information.

**US-12**  
As a poster, I want to mark my listing as resolved once the item has been returned so that it is removed from the active board.
